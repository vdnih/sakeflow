# sakeflow ブログ PRD（プロダクト要件定義）

> バージョン: 1.0  
> 最終更新: 2026-04-29

---

## ブログコンセプト

**「お酒と旅を楽しむためのブランドサイト」**

sakeflow ブログは、お酒と旅に関する情報を発信するコンテンツサイトです。  
sakeflow-log アプリへの導線となり、ブランドの認知向上を担います。

---

## ターゲット

- お酒（日本酒・ワイン・ウイスキー等）に興味がある方
- 旅先でのお酒体験に関心がある方
- sakeflow-log アプリの潜在ユーザー

## コンテンツ管理（microCMS）

### コンテンツ型

#### Article（ブログ記事）

| フィールド | 型 | 説明 |
|---------|---|------|
| title | string | 記事タイトル |
| content | richtext | 記事本文（HTML） |
| thumbnail | image | サムネイル画像 |
| category | reference（Category） | カテゴリ |
| writer | reference（Writer） | ライター |
| description | string | 記事説明（OGP 用） |

#### Category

| フィールド | 型 | 説明 |
|---------|---|------|
| name | string | カテゴリ表示名 |
| value | string | カテゴリ識別値（スラッグ） |

#### Writer

| フィールド | 型 | 説明 |
|---------|---|------|
| name | string | ライター名 |
| profile | string | プロフィール文 |
| image | image | プロフィール画像 |

---

## URL 設計

| パス | 内容 |
|-----|------|
| `/` | トップページ（最新記事一覧） |
| `/articles/[id]` | 記事詳細 |
| `/categories/[slug]` | カテゴリ別一覧 |
| `/search` | 記事検索（Phase 2） |

---

## 非機能要件

| 要件 | 目標値 |
|-----|-------|
| LCP（Core Web Vitals） | 2.5 秒以内 |
| SEO | 全ページ meta description あり |
| OGP | 全記事 OGP 設定 |
| レスポンシブ | スマートフォン・タブレット・PC 対応 |

---

## スコープ外

- ユーザー投稿・コメント機能
- 有料コンテンツ・課金機能
- ライブ配信・動画埋め込み（外部プレイヤー経由は許容）
