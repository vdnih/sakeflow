## アプリコンセプト

**「お酒をもっと楽しむためのアプリ」**

sakeflow-log は、お酒との素敵な出会いと思い出を記録するためのアプリです。  
日々の飲酒体験をより豊かにし、あなたの "お酒ライフ" をサポートします。

---

## ターゲットユーザー

- 日本酒・ワイン・ウイスキーなど多様なお酒を楽しみたい方
- 飲んだお酒を記録・管理したい方
- 新しいお酒との出会いを求めている方

---

## 提供価値

- お酒の記録を通じて、より深く自分の好みを知る
- 新しいお酒や飲み方の発見
- お酒好き同士の交流や情報共有

## ルーティング設計（Web 実装）

> 現在の実装は Web 対応のみ。将来的に iOS/Android 対応予定（Phase 3）。

- 画面ごとに明示的なパスを用意し、拡張性・保守性を高める
- 認証必須画面には `AuthGuard` を導入
- `/` ルートは認証状態に応じて `/login` または `/home` にリダイレクト

### 実装済みルーティング（`app/lib/main.dart`）

```dart
routes: {
  '/login': (context) => SignInScreen(...),
  '/home': (context) => const HomeScreen(),
  '/ai-label': (context) => const AiLabelScreen(),
}
```

## 多様なお酒への対応

本アプリは日本酒だけでなく、以下のお酒カテゴリに対応します：

| カテゴリ（日本語）| カテゴリ（英語）|
|----------------|--------------|
| 日本酒 | sake |
| ワイン | wine |
| ビール | beer |
| ウイスキー | whisky |
| 焼酎 | shochu |
| リキュール | liqueur |
| ブランデー | brandy |
| ジン | gin |
| ウォッカ | vodka |
| ラム | rum |
| その他 | other |

すべての機能（記録・検索・AI 判別・履歴）で `category_name` フィールドによるフィルタ・表示切替が可能です。
