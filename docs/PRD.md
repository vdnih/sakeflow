## 非機能要件

| 要件 | 目標値 |
|-----|-------|
| blog 初期表示速度 | LCP 2.5 秒以内 |
| app 認証完了時間 | 3 秒以内 |
| AI ラベル認識応答 | 10 秒以内（Cloud Functions コールドスタート含む） |
| app 対応ブラウザ | Chrome / Safari 最新版 |
